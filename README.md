# enjoyingredients

## Prerequisites

- Docker is installed

## Local development setup

``` bash
$ docker-compose run --rm node
$ docker-compose up -d
$ ./install-db.sh
```

The website is running on ``localhost:3000``. \
Directus is accessible via ``localhost:8090``. \
The database is accessible via ``localhost:3306``.
